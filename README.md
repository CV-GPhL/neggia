Neggia
===========

XDS plugin that reads hdf5 files written by Dectris Eiger Detectors

## Build
### Prerequisites
* gcc-4.8
* cmake 3.6
* DectrisHdf5 (https://github.com/dectris/DectrisHdf5.git)
### Building
* mkdir build
* cd build
* cmake ..
* make
The plugin file is found in src/dectris/neggia/plugin/dectris-neggia.so
